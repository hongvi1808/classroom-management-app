# Backend: cma-backend
## Install 
1. ### NodeJS
2. ### Yarn
```bash
npm install --global yarn
```
## Run project
1. ### Install node_module
```bash
yarn install
```
2. ### Add info Twilio (Github don't allow to push that info)
```bash
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_FROM_TO=
```
3. ### Run project
#### for .env.example file
```bash
yarn start
```

#### for .env (for developer)
```bash
yarn start:dev
```
